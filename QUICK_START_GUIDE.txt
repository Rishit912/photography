ğŸ› TROUBLESHOOTING

Q: Client key not working?
A: 
   - Check key is exact match from Admin Dashboard
   - Try incognito mode (clear old sessions)
   - Ensure you added a client first

Q: Images not showing?
A:
   - Verify URL is accessible in browser address bar
   - Try test image: https://picsum.photos/seed/test/800/600
   - Check no typos in URLs

Q: CSS looks broken?
A:
   - Press Ctrl+Shift+Delete (clear browser cache)
   - Restart dev server (Ctrl+C, then npm run dev)
   - Check all imports in src files

Q: Server won't start?
A:
   - Port 5174 might be in use
   - Run: netstat -ano | findstr :5174 (check process)
   - Kill process or use different port
